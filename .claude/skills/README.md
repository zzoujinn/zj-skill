# Skills 目录

运维开发工程师常用的 Claude Code Skills 集合。

## Skill 列表

| Skill | 说明 |
|-------|------|
| **ansible-playbook** | Ansible 自动化编排。编写 Playbook、Role、Inventory，实现批量部署、配置、巡检等运维任务自动化。 |
| **ci-cd-pipeline** | CI/CD 流水线配置。支持 GitLab CI、GitHub Actions、Jenkins，涵盖构建、测试、部署全流程。 |
| **code-review** | 通用代码审查。检查代码质量、安全性、性能和最佳实践。 |
| **data-analysis** | 数据分析与可视化。支持 CSV、JSON、Excel 等格式，生成图表和统计报告。 |
| **devops-code-review** | 运维代码专项审查。针对 Shell、Python、Go 运维脚本，关注部署安全和运维最佳实践。 |
| **disk-performance-diagnosis** | 磁盘性能诊断。分析磁盘使用情况、I/O 性能，识别瓶颈并提供优化建议。支持远程诊断。 |
| **docker-helper** | Docker 容器化辅助。编写/优化 Dockerfile 和 docker-compose，多阶段构建、镜像瘦身、容器排查。 |
| **k8s-troubleshooting** | Kubernetes 故障排查。排查 Pod 异常、节点问题、网络故障、存储问题，审查 YAML 配置。 |
| **log-analysis** | 日志分析与故障排查。分析应用/系统/中间件日志，定位错误根因，统计错误趋势。 |
| **meeting-notes** | 会议记录。创建、格式化和组织会议纪要，管理行动项和跟进事项。 |
| **nginx-config** | Nginx 配置生成与审查。反向代理、负载均衡、SSL/TLS、限流、缓存、安全和性能优化。 |
| **prometheus-grafana** | 监控告警配置。编写 PromQL 查询、配置告警规则、设计 Grafana 仪表盘、配置 Alertmanager。 |
| **shell-toolkit** | Shell 脚本工具箱。生成备份/清理/监控/巡检/批量执行等运维脚本，提供编写规范。 |
| **vue-devops-platform** | Vue 3 运维平台开发。基于 Element Plus 的运维平台页面生成和编码规范。 |
